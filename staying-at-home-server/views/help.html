1. Download the app to your mobile to start playing

  2. Enable GPS location from configuration menu

  3. First Tree will start growing, after 6 consecutive hours without leaving home the Tree will be added to your forest

  4. GPS monitor will track if you leave home, and in that case the current growing Tree will be deleted

  5. A new Tree will start growing once you come back home

  6. After getting 10 Trees you will be moved to the next level

As more people start using the App, data will be available on more cities.
