Since not all countries have data at this point, please follow steps below to use "Mexico -> Jalisco -> Ciudad Guzmán" to visualize data on the map:

1. When you first load the web page you’ll land on the “countries” tab. On the left there’s a menu with country names from which you can select the one you’re interested in. Once you click it, if available, the map will be shown in the right panel. Please Select Mexico, you can use the search field to find it faster

2. After you select a country, you can move into the states tab (from the upper menu) and select a state from the left pane; this menu is loaded based on which country was selected in the previous tab. Same as with countries: once you click the state you’re interested in, if available, the map will be shown in the right panel. Please select Jalisco and the corresponding map will be displayed.

3. Once a country and state are selected, you can move into the cities tab (from the upper menu) and select a city from the left pane; this menu is loaded based on which state was selected in the previous tab. Same as with countries and states: once you click the city you’re interested in, if available, the map will be shown in the right panel. Plase select Ciudad Guzmán and the corresponding map will be displayed.
